# 🌍 GeoTravel

Веб-приложение для планирования путешествий, расчёта расстояний между городами и сохранения маршрутов. 
Разработано как индивидуальный проект на Python (Flask).

## 🛠 Стек технологий
- **Backend:** Python 3.11, Flask 2.x, Flask-Login
- **База данных:** SQLite3, SQLAlchemy ORM
- **Фронтенд:** HTML, CSS, Bootstrap 5, Jinja2
- **API:** Yandex Geocoder (геокодинг), Haversine formula (расчёт расстояний)

## 📦 Требования
- Python 3.8+
- `pip install -r requirements.txt`

## 🚀 Запуск
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите сервер:
   ```bash
   python main.py
   ```
3. Откройте в браузере: `http://127.0.0.1:8080/`

## ✨ Реализованный функционал
- Регистрация и авторизация пользователей
- Создание путешествий с добавлением остановок (города)
- Автоматическое определение координат через Yandex Geocoder API
- Расчёт расстояний между точками маршрута
- Загрузка и отображение фотографий маршрутов
- Личный профиль с историей путешествий
- REST API для получения данных о расстояниях и маршрутах
